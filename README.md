# Angular_NodeJS_SCM_Frame

Angular SCM Frame

**Run NodeJS App**

1. create .env copy from env-example
2. cd nodejs-frame
3. npm install
4. npm run dev

**Run Angular App**

1. cd angular-frame
2. npm install
3. npm run start
